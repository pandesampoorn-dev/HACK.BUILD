\# HACK.BUILD  
Find your team. Build the future.
HACK.BUILD is a team-matching and project collaboration platform for developers, featuring authentication, project management, and real-time collaboration workflows.



\## Tech Stack
\- React + Vite
\- Spring Boot
\- MySQL
\- REST APIs



\## Project Structure
HACK.BUILD/

├── hackbuild-frontend/ # React client

└── hackbuild-backend/ # Spring Boot API



\##  Setup
\### Backend

cd hackbuild-backend
./mvnw spring-boot:run



\### Frontend

cd hackbuild-frontend
npm install
npm run dev

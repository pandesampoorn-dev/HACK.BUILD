\# HACK.BUILD  



Find your team. Build the future.



\## Tech Stack



\- React + Vite

\- Spring Boot

\- MongoDB

\- REST APIs





\## Project Structure



HACK.BUILD/

├── hackbuild-frontend/ # React client

└── hackbuild-backend/ # Spring Boot API



\##  Setup



\### Backend



cd hackbuild-backend

./mvnw spring-boot:run







\### Frontend



cd hackbuild-frontend

npm install

npm run dev
